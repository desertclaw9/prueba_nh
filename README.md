## Installation

1. Clone repository
   ```sh
   git clone ..
   ```
   
2. Create virtual environment
   ```sh
   cd prueba_nh
   python3 -m venv venv
   source venv/bin/activate
   pip install --upgrade pip
    ```

3. Install requirements
    ```sh
    pip install -r requirements.txt
    ```
